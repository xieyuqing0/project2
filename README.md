# project2
实践周任务
一、毕设推进：
在毕设“数据驱动的铁电陶瓷电滞回线模型构建”中，神经网络模型BiGRU+Multi-Head Attention在实验二“基于有限温度点的跨温区电滞回线预测”中能学习到铁电陶瓷极化强度随温度的变化曲线，展现出一定的泛化性能，但对比实验一“基于小样本的电滞回线内插预测”表现较差，实验二中预测值与实验值的决定系数R2较实验一小，归一化均方误差较实验一大，实践周进行了模型的超参数调优，将实验二的精度提高
二、文献研读：
因为我的毕设跟深度学习强相关，所以研读了推荐书目中的Ian Goodfellow等人编写的《深度学习》，也就是业内常说的花书，读完之后对深度学习的理解又有了进一步的提升，这本书没有堆砌零散的实操技巧，而是从根源讲清楚了深度学习为什么有效、核心逻辑是什么，读完之后受益匪浅，帮助我跳出单纯调参的局限，真正读懂了算法背后的底层逻辑。
